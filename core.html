Golang binary support (references)

Type plugins (e.g. PE, ELF, Mach-O) use the following references for Go symbol and runtime parsing:

  • Go runtime2

  • gosym/pclntab

  • go_parser pclntbl

  • golang_loader_assist

  • Mandiant: Golang internals - symbol recovery
